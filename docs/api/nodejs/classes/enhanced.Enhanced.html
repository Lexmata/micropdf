    Class Enhanced

    Enhanced NanoPDF API for high-level PDF operations.

    Provides convenient methods for common PDF manipulation tasks that would otherwise require multiple lower-level API calls.

    Enhanced

    Example

    const enhanced = new Enhanced();
    const doc = Document.open('document.pdf');

    // Add a watermark to all pages
    for (let i = 0; i < doc.pageCount; i++) {
      enhanced.addWatermark(
        doc,
        'DRAFT',
        { x: 300, y: 400 },
        48,
        0.2
      );
    }

    doc.save('watermarked.pdf');
    doc.close();
