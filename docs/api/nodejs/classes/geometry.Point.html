    Class Point

    A 2D point with floating-point coordinates.

    Points are immutable - all operations return new Point instances rather than modifying the existing point. This makes them safe to use in functional programming contexts and prevents accidental mutations.

    Point

    Implements

    Example

    // Create a point
    const p1 = new Point(10, 20);

    // Transform it
    const p2 = p1.scale(2); // Point(20, 40)

    // Calculate distance
    const distance = p1.distanceTo(p2); // 22.36...

    // Points are immutable
    const p3 = p1.add(new Point(5, 5)); // p1 is unchanged

      Normalizes this point to unit length.
