    Interface TextSpan

    A text span representing a run of text with consistent formatting.

    Text spans group glyphs that share the same font, size, and rendering properties. They are the basic unit of text layout in PDF documents.

    TextSpan

    Example

    const span: TextSpan = {
      font: 'Arial',
      size: 12,
      wmode: 0, // Horizontal
      trm: Matrix.identity(),
      items: [
        { x: 10, y: 20, glyph: 65, unicode: 0x0041 }, // 'A'
        { x: 20, y: 20, glyph: 66, unicode: 0x0042 }  // 'B'
      ]
    };
